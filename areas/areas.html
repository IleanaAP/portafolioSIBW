Aprende a obtener el área de figuras

La página donde aprenderás a cómo obtener el área de tres figuras geométricas: rectángulo, triángulo y círculo.

Área de un rectángulo

El área de un rectángulo se calcula a partir de los dos lados diferentes (a y b). Es el producto o multiplicación de los dos lados juntos del rectángulo.

Área = a * b

siendo a y b los dos lados diferentes

Esta fórmula también podría obtenerse de la fórmula del área del paralelogramo, pero por ahorita no la veremos. Si la base del rectángulo es uno de sus lados (en este caso b, la altura relativa a la base será el lado a, y aplicandola fórmula anterior obtendríamos la del área del rectángulo.


Ejemplo

Sea un rectángulo cuyos lados son iguales dos a dos de longitud a = 3 cm. y b = 5 cm.

El área del rectángulo será el producto de los dos lados diferentes, es decir:

Área = a * b = 3 * 5 = 15 cm2

El área del rectángulo de lados a = 3 cm. y b = 5 cm. es de 15 cm2.

Área de un triángulo

El área de un triángulo se calcula por diferentes procedimientos según el tipo de triángulo de que se trate o de los elementos que se conozcan de ese triángulo. Aquí veremos la fórmula general para calcular el área de un triángulo.

La fórmula general para calcular el área de un triángulo es:

Área = (b * h) / 2

donde b es la base y h es la altura

Esta fórmula se utiliza cuando se conocen las medidas de un lado (b) y la altura (h) asociada a dicho lado. Ese lado ejerce como base.


Ejemplo

Sea un triángulo rectángulo con los lados que forman el ángulo recto (a y b) conocidos, siendo a = 3 cm. y b = 4 cm.

¿Cuál es su área?

Aplicamos la fórmula general:

Área = (b * a) / 2 = (4 * 3) / 2 = 12 / 2 = 6 cm2

Y se obtiene que su área es de 6 cm2.

Área de un círculo

El área de un círculo es igual al producto o multiplación de π (pi) por el radio (r) al cuadrado.

Área = π * r2 = π * (D2 / 4)

siendo r el radio y D el diámetro del círculo

También se puede calcular el área conociendo el diámetro del círculo (D), ya que éste es el doble del radio.

Como un círculo es un polígono regular de infinitos lados, podemos aplicarla fórmula general del área del polígono regular:

Área = (Perímetro * apotema) / 2 =

= (2 * π * r)* r / 2 = (2 * π * r2) / 2 = π * r2

Ejemplos

Círculo y su radio

Sea un círculo de radio conocido, siendo éste r = 2 cm.

Aplicamos la fórmula anterior:

Área = π * r2 = π * 22 = 12.57 cm2

Y se obtiene que el área de un círculo de radio 2 cm. es de 12.57 cm2.

Círculo y su diámetro

Ahora supongamos que tenemos un círculo de diámetro conocido D = 5 cm.

¿Cuál es su área?

Área = π * (D2 / 4) = π * (52 / 4) = 19.63 cm2

En este caso, el área de un círculo de diámetro 5 cm. es de 19.63 cm2.
